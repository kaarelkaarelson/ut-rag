There will be weekly homework involving HTML5, JavaScript or PHP. Students are expected to have experience in web application development. Grading Final grade structure: Homework: 70% Final test: 30% Schedule 0. Introduction 1. Web, HTTP protocol, HTTPS, Cookies 2. Cross-Site Scripting (XSS) 3. Cross-Site Request Forgery (CSRF) 4. Same-Origin Policy (SOP) 5. Authentication and Session Management 6. SQL Injection 7. Server-Side Vulnerabilities 8.
