Participants are expected to write an overview article on a chosen topic, give oral presentations, and actively participate in all the seminars. Some of the topics covered by the seminar are: Internet of Things Location-based services Mobile web services Cloud applications and their development Migrating existing applications to the cloud Cloud-based software frameworks Cloud performance MapReduce and large-scale data processing in the cloud Scientific computing in the cloud Mobile applications User interfaces for mobile devices Android development iOS development To pass the seminar, the student has to : Give an oral presentation on a chosen topic Participate actively in all the seminars Write a report on a chosen topic Provide peer review Additional materials Refer to previous semesters: 2023/2024 fall materials 2022/2023 spring materials 2022/2023 fall materials 2021/2022 spring materials 2021/2022 fall materials Arvutiteaduse instituut Loodus- ja täppisteaduste valdkond Tartu Ülikool Tehniliste probleemide või küsimuste korral kirjuta: Kursuse sisu ja korralduslike küsimustega pöörduge kursuse korraldajate poole.
