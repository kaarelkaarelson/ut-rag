Loomuliku keele töötlus - Kursused - Arvutiteaduse instituut Kursused 2023/24 kevad Loomuliku keele töötlus (LTAT.01.001) EN Logi sisse Loomuliku keele töötlus 2023/24 kevad HomePage Schedule Project Seminars Homeworks Theory test FAQ Theory test Theory test will take place on 05.04.2023 via Moodle. No pyshical presence in class is required. The time for taking the test will be limited, but the time window when the test is available will be longer. The test will contain both multiple-choice and open-ended questions. There will be likely questions that require working on some toy examples. Thus make sure that you have a pen, a paper and a calculator in reach. The topics for the theory test are available here .
