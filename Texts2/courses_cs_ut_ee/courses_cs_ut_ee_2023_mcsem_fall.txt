Description: Cloud and Mobile computing technologies have become an important part of scientific and infotechnology development in recent years and this seminar aims to introduce students to newest concepts and advances in these fields. Participants are expected to write an overview article on a chosen topic, give oral presentations and take active part of all the seminars. Some of the topics covered by the seminar are: Internet of Things Location based services Mobile web services Cloud applications and their development Migrating existing applications to the cloud Cloud based software frameworks Cloud performance MapReduce and large scale data processing in cloud Scientific computing in the cloud Mobile applications User interfaces for mobile devices Android development iOS development To pass the seminar, student has to : Give an oral presentation on a chosen topic Participate actively in all the seminars Write a report on a chosen topic Provide peer review Additional materials Refer to previous semesters: 2022/2023 spring materials 2022/2023 fall materials 2021/2022 spring materials 2021/2022 fall materials Attach:logo3.jpeg Δ Arvutiteaduse instituut Loodus- ja täppisteaduste valdkond Tartu Ülikool Tehniliste probleemide või küsimuste korral kirjuta: Kursuse sisu ja korralduslike küsimustega pöörduge kursuse korraldajate poole.
